# elias-berg.github.io

The base code for Eli's personal site, hosted at [elias-berg.com](https://elias-berg.com).

## Getting Started

To run the static site locally, you'll need `npx` installed. Then run the following command:

`npx browser-sync start --server`

Note this will prompt you to install `browser-sync` if you haven't already.

## Hosting

The public domain, elias-berg.com, uses CloudFlare DNS to route to to the statically hosted content by GitHub pages.
